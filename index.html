Descrição: Este projeto consiste em um webcrawler automatizado capaz de extrair dados de websites específicos e armazená-los em um banco de dados PostgreSQL. O crawler é configurável para diferentes fontes de dados e possui uma interface administrativa para monitoramento e ajustes.
